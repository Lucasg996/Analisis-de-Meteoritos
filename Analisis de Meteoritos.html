Parte 1

Pregunta: Cul es la distribucin geogrfica de los meteoritos y donde cayeron los ms pesados?

Introduccin: La cada de meteoritos ha sido un fenmeno de inters cientfico y cultural a lo largo de la historia. La distribucin geogrfica y las caractersticas fsicas de estos cuerpos celestes nos proporcionan informacin valiosa sobre su origen, trayectoria y las condiciones de impacto en la Tierra. Este anlisis busca responder a dos preguntas fundamentales: Dnde se distribuyen geogrficamente los meteoritos y en qu regiones se encuentran los de mayor masa? Para ello, utilizamos una base de datos proporcionada por la NASA, que contiene ms de 45.000 registros, incluyendo detalles como el nombre, la masa, el ao de impacto y la ubicacin geogrfica (latitud y longitud) de cada meteorito. Esta base de datos es valiosa para entender la distribucin de meteoritos en nuestro planeta. En respuesta a las preguntas anteriormente mencionadas, se utilizarn las variables de masa, latitud y longitud del conjunto de datos. Se explorar cmo la masa de los meteoritos vara en diferentes ubicaciones y se identificarn las reas con meteoritos ms grandes.

Enfoque: Empleamos herramientas del paquete ggplot2 de R para visualizar los datos de manera clara y comprensible. Para ello, se disearon dos tipos de grficos:

Grfico de dispersin: Representa cada meteorito en funcin de su ubicacin geogrfica, usando coordenadas de latitud y longitud. El tamao y el color de los puntos destacan la masa total de los meteoritos, diferenciando entre aquellos que superan un umbral de masa establecido (meteoritos grandes) y los que no (meteoritos pequeos).El uso de un grfico de dispersin es apropiado aqu porque facilita la identificacin de patrones de ubicacin en relacin con la masa.

Mapa de calor: Muestra la densidad de impactos de meteoritos en distintas regiones del mundo. Este grfico destaca las zonas de mayor concentracin geogrfica mediante un gradiente de color. Este enfoque es til para identificar patrones en la distribucin de cadas registradas, revelando zonas con alta densidad de impactos y otras con menor frecuencia de registros.

Analisis:

# Define el umbral de masa para clasificar los meteoritos grandes
umbral_masa <- 1e+07

# Agrupa los meteoritos por latitud y longitud y calcula la masa total
meteoritos_agrupados <- meteoritos %>%
  group_by(lat, long) %>%
  summarise(total_masa = sum(mass, na.rm = TRUE), .groups = 'drop') %>%
  # Filtra filas con NA en lat o long
  filter(!is.na(lat) & !is.na(long))

# Crea el mapa base y superpone los puntos de meteoritos
ggplot() +
  # Agrega el mapa base con contornos de pases
  borders("world", colour = "gray70", fill = "gray95") +
  # Superpone los puntos de meteoritos, con tamao y color basados en la masa
  geom_point(data = meteoritos_agrupados, 
             aes(x = long, y = lat, size = total_masa, color = total_masa > umbral_masa), 
             alpha = 0.7) +
  # Ajusta la paleta cromtica para los meteoritos
  scale_color_manual(
    values = c("FALSE" = "#1f77b4", "TRUE" = "red"),  # Azul profundo y dorado
    labels = c("FALSE" = "Meteoritos pequeos", "TRUE" = "Meteoritos grandes")
  ) +
  scale_size_continuous(range = c(1, 10), guide = guide_legend(title = "Tamao de masa")) +
  labs(
    title = "Distribucin geogrfica de meteoritos",
    subtitle = "El tamao y el color representan la masa total de los meteoritos",
    x = "Longitud",
    y = "Latitud",
    color = "Clasificacin de masa",
    caption = "Fuente: NASA"
  ) +
  theme_minimal() +
  theme(
    plot.title = element_text(face = "bold", size = 16, color = "#333333"),
    plot.subtitle = element_text(size = 12, color = "#555555"),
    plot.caption = element_text(hjust = 0.5, color = "#666666"),
    axis.text = element_text(color = "#333333"),
    axis.title = element_text(face = "bold", color = "#333333"),
    legend.position = "right",  # Cambia la posicin de la leyenda
    legend.title = element_text(face = "bold"),
    legend.text = element_text(size = 10)
  )

# Cargar datos de meteoritos (modifica segn tu nombre de dataframe y columnas)
meteoritos_limpios <- meteoritos %>%
  filter(!is.na(lat) & !is.na(long))  # Elimina filas con latitud o longitud faltantes

# Crear el mapa de calor
ggplot() +
  # Mapa base de los continentes
  borders("world", colour = "gray70", fill = "gray95") +
  # Crear mapa de calor usando stat_density_2d
  stat_density_2d(
    data = meteoritos_limpios,
    aes(x = long, y = lat, fill = after_stat(level)),
    geom = "polygon",
    contour = TRUE,
    alpha = 0.8  # Transparencia para mejor integracin con el mapa base
  ) +
  # Escala de color personalizada basada en la paleta del primer grfico
  scale_fill_gradient(
    low = "#1f77b4",  # Azul profundo
    high = "red",     # Rojo intenso
    guide = guide_colorbar(title = "Densidad")
  ) +
  labs(
    title = "Mapa de Calor de la Cada de Meteoritos",
    subtitle = "Distribucin geogrfica basada en la densidad de impactos",
    x = "Longitud",
    y = "Latitud",
    caption = "Fuente: NASA"
  ) +
  theme_minimal() +
  theme(
    plot.title = element_text(face = "bold", size = 16, color = "#333333"),
    plot.subtitle = element_text(size = 12, color = "#555555"),
    axis.title = element_text(face = "bold", size = 12, color = "#333333"),
    axis.text = element_text(size = 10, color = "#333333"),
    legend.position = "right",  # Mueve la leyenda al lado derecho
    legend.title = element_text(face = "bold", size = 10, color = "#333333"),
    legend.text = element_text(size = 9, color = "#333333"),
    plot.caption = element_text(hjust = 0.5, color = "#666666")
  )

Discusin: El mapa de calor revela que los meteoritos tienden a concentrarse en reas especficas, destacndose regiones de frica, Europa, Asia y la Antrtida. Esta distribucin no solo refleja los patrones de impacto, sino tambin la influencia de la actividad humana en la exploracin, registro y recuperacin de meteoritos, que es ms frecuente en zonas accesibles y de mayor inters cientfico. Por otro lado, las regiones menos pobladas o de difcil acceso, como los ocanos y reas remotas, presentan una menor densidad de registros. Esto no implica necesariamente que caigan menos meteoritos en esas zonas, sino que es menos probable que sean detectados y documentados, lo que genera un sesgo en los datos disponibles. Finalmente, la dispersin geogrfica de los meteoritos de gran tamao indica que no existe una correlacin clara entre la masa de los meteoritos y las regiones donde impactan. Su distribucin aleatoria sugiere que los factores naturales, como la trayectoria y la velocidad, tienen mayor influencia en la ubicacin de estos impactos que cualquier caracterstica geogrfica especfica.

Parte 2

Pregunta: La clase tienen alguna distribucion temporal?

Introduccin: La cada de meteoritos no solo despierta inters por su impacto fsico, sino tambin por las tendencias que se pueden observar a lo largo del tiempo. Analizar su distribucin temporal nos proporciona informacin valiosa sobre cambios en la frecuencia de impactos, avances tecnolgicos en su deteccin y factores histricos que influyen en su registro. En respuesta a esta pregunta, se analizar la cantidad de meteoritos registrados en intervalos de dos aos para identificar tendencias a corto plazo, y se examinarn las variaciones en las principales clases de meteoritos por dcada para obtener una perspectiva histrica ms amplia.

Enfoque: Para explorar la distribucin temporal, utilizamos dos enfoques principales:

Tendencia bianual: Se agruparon los meteoritos por intervalos de dos aos desde 1960 hasta 2012. Se visualiz la cantidad de meteoritos registrados en cada intervalo mediante un grfico de lneas, destacando posibles tendencias de aumento o disminucin en el tiempo.

Distribucin de clases por dcada: Se analizaron las clases de meteoritos ms observadas en cada dcada desde 1900. Se seleccionaron las diez clases con mayor nmero de registros y se representaron mediante un grfico de barras apiladas, permitiendo observar cmo vara la composicin de las clases a lo largo de las dcadas.

Analisis:

# Filtrar los datos y calcular el intervalo de dos aos
meteoritos_filtrados <- meteoritos %>%
  filter(!is.na(year) & year >= 1960 & year <= 2012) %>%
  mutate(intervalo = floor(year / 2) * 2)

# Contar la cantidad de meteoritos por intervalo de dos aos
meteoritos_resumen <- meteoritos_filtrados %>%
  group_by(intervalo) %>%
  summarise(cantidad = n(), .groups = 'drop')

# Crear el grfico de lneas
ggplot(meteoritos_resumen, aes(x = intervalo, y = cantidad)) +
  geom_line(color = "steelblue", linewidth = 1) + 
  geom_point(color = "black", size = 2) +
  scale_x_continuous(
    name = "Ao (Intervalo de 2 aos)", 
    breaks = seq(1960, 2012, by = 2),
    limits = c(1960, 2012)
  ) +
  scale_y_continuous(
    name = "Cantidad de Meteoritos",
    limits = c(0, max(meteoritos_resumen$cantidad, na.rm = TRUE))
  ) +
  labs(title = "Tendencia de Meteoritos Registrados cada 2 aos") +
  theme_minimal() +
  theme(
    plot.title = element_text(face = "bold", size = 16, hjust = 0.5),
    axis.text.x = element_text(angle = 45, hjust = 1, size = 8),
    axis.text.y = element_text(size = 10),
    axis.title.x = element_text(face = "bold"),
    axis.title.y = element_text(face = "bold")
  )

# Filtrar los datos y calcular la dcada
meteoritos_filtrados <- meteoritos %>%
  filter(!is.na(year) & year >= 1900 & year < 2100) %>%
  mutate(decada = floor(year / 10) * 10)

# Contar el nmero de meteoritos por dcada y clase
meteoritos_class <- meteoritos_filtrados %>%
  group_by(decada, class) %>%
  summarise(count = n(), .groups = "drop")

# Seleccionar solo las clases con ms observaciones (top 10)
top_classes <- meteoritos_class %>%
  group_by(class) %>%
  summarise(total_count = sum(count), .groups = "drop") %>%
  top_n(10, total_count) %>%
  pull(class)

# Filtrar los datos para incluir solo las clases principales
meteoritos_top_classes <- meteoritos_class %>%
  filter(class %in% top_classes)


# Crear una paleta de colores personalizada con tonalidades ms suaves de azul y rojo
colores_personalizados <- c(
  "#1f77b4",  # Azul profundo
  "red",  # Rojo intenso
  "#9ac9e6",   # Azul claro
  "#e68985",   # Rojo claro
  "#7ea1cd",   # Azul tenue
  "#e56a59",   # Rojo anaranjado
  "#aad4f5",   # Azul muy claro
  "#ffb3ab",   # Rosa plido
  "#a8c9e7",   # Azul plido
  "#f1c1b7"    # Rosa suave
)

# Crear el grfico de barras agrupado por dcada
ggplot(meteoritos_top_classes, aes(x = factor(decada), y = count, fill = class)) +
  geom_bar(stat = "identity", position = "stack") +
  labs(
    title = "Distribucin de las Clases Principales de Meteoritos",
    subtitle = "Top 10 clases de meteoritos con ms observaciones desde 1800",
    x = "Dcada",
    y = "Nmero de Observaciones",
    fill = "Clase de Meteorito"
  ) +
  scale_fill_manual(values = colores_personalizados) + # Usar la paleta de colores personalizada
  theme_minimal() +
  theme(
    plot.title = element_text(face = "bold", size = 16, hjust = 0.5, color = "#333333"),
    plot.subtitle = element_text(size = 12, hjust = 0.5, color = "#555555"),
    axis.text.x = element_text(angle = 45, hjust = 1, size = 9, color = "#333333"),
    axis.text.y = element_text(size = 10, color = "#333333"),
    axis.title.x = element_text(face = "bold", color = "#333333"),
    axis.title.y = element_text(face = "bold", color = "#333333"),
    legend.position = "right", # Mover la leyenda al lado derecho
    legend.title = element_text(face = "bold", size = 10, color = "#333333"),
    legend.text = element_text(size = 9, color = "#333333")
  )

Discusin: Los resultados destacan una clara tendencia temporal en el registro de meteoritos. El incremento en los registros a partir de los aos 60 se alinea con avances tecnolgicos y cientficos, indicando que ms meteoritos estn siendo documentados, no necesariamente que caigan ms en esa poca. Por otro lado, la estabilidad relativa en las clases dominantes a lo largo de las dcadas sugiere que la composicin general de los meteoritos que llegan a la Tierra no ha variado significativamente en trminos de sus caractersticas principales. Sin embargo, las variaciones observadas en proporciones podran reflejar prioridades de investigacin o descubrimientos excepcionales en ciertos perodos. Estos hallazgos subrayan la importancia de considerar tanto los factores naturales como los histricos y tecnolgicos en la interpretacin de los datos temporales de los meteoritos.
